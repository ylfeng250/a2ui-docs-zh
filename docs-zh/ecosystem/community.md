# 社区

欢迎加入 A2UI 社区！该项目正在共同构建智能体驱动界面的未来。

## 参与进来

A2UI 是一个在 Apache 2.0 许可下开源的项目。欢迎开发者、研究人员以及任何对推进智能体用户界面感兴趣的人做出贡献。

## 社区展示

用 A2UI 构建了什么？展示出来！

分享作品的最佳方式是在 **[GitHub Discussions](https://github.com/google/A2UI/discussions)** 中发帖，附上简短描述、代码链接和演示视频（4 分钟以内）。值得关注的项目会在文档和发行说明中展示。

**社区构建的渲染器**列在 [生态系统渲染器](renderers.md) 页面上 — 在那里查看社区实现以及添加您自己的说明。

仓库中的**官方示例**是优质 A2UI 项目的绝佳参考：

- [Restaurant Finder](https://github.com/google/A2UI/tree/main/samples/agent/adk/restaurant_finder) — 带有动态表单的 ADK Agent。
- [Contact Lookup](https://github.com/google/A2UI/tree/main/samples/agent/adk/contact_lookup) — 带数据绑定的搜索 UI。
- [Angular 客户端](https://github.com/google/A2UI/tree/main/samples/client/angular) — 画廊 + 餐厅客户端。
- [Lit 外壳](https://github.com/google/A2UI/tree/main/samples/client/lit/shell) — 最小化的 Web 组件客户端。

## 项目合作伙伴

A2UI 与 Google Opal、Flutter（GenUI SDK）、Gemini Enterprise、AG UI / CopilotKit 和 A2A 团队联合开发。有关详细信息和案例研究，请参阅 [A2UI 的世界](a2ui-in-the-world.md)。

如果您为 A2UI 做出了重要贡献，请在此列表中添加您的名字。

## 行为准则

A2UI 致力于为每个人提供一个友好和包容的环境。所有参与者应：

- 尊重和体谅他人。
- 欢迎新用户并帮助他们入门。
- 以社区的最大利益为重。
- 对他人展现同理心。

不可接受的行为将不被容忍。请向项目维护者报告问题。

## 认可

所有贡献都受到赞赏！贡献者在以下方面获得认可：

- GitHub 贡献者图表。
- 发行说明。
- 项目致谢。

重大贡献者可能会被邀请加入项目的维护者团队。

## 保持更新

了解项目动态：

- **Watch GitHub 仓库**以获取通知。
- **Star 仓库**以收藏和表示支持。
- **关注发布**以获取新版本的通知。

## 贡献方式

**[github.com/google/A2UI](https://github.com/google/A2UI)**

- **报告问题**：发现了 bug？[提 Issue](https://github.com/google/A2UI/issues)。
