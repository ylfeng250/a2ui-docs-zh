# A2UI 中文文档

A2UI 协议的中文翻译文档。

[英文原文](https://github.com/google/A2UI) | [在线文档](https://google.github.io/a2ui-docs-zh/)

[中文翻译](https://ylfeng250.github.io/a2ui-docs-zh/)

## 同步上游文档

```bash
./sync-docs.sh
```

## 本地开发

```bash
npm install
npm run docs:dev
```

## 构建

```bash
npm run docs:build
```
